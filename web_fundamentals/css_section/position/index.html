CSS Position

Position Rule #1 - Do Not Abuse Position!

Five values for the position property:

  1. Static
  2. Relative
  3. Absolute
  4. Fixed
  5. Sticky

Some good uses of the position property include:

  • Breaking the plane of a container with another element.
  • Navbar that always remains in place (at the top of your window)
  • Cookie notices that remain fixed

Static

The default value of position. Left, top, right, and bottom will NOT move your element.

Relative

The element is positioned relative to its normal position in the document flow. Left, top, right, and bottom will move your element.

Absolute

The element is positioned relative to its closest positioned ancestor. Left, top, right, and bottom will move your element. When you absolutely position an element, you must relatively position its parent.

Fixed

The element is positioned relative to the window. The cookie box element at the bottom of this page is fixed.
