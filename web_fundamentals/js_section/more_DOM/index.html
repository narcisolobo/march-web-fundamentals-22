Document Object Model

The representation or "model" of the document (browser window) as an object.

Event Listeners

Event listeners are DOM methods that "listen" for events.

  • Mouse Clicks
  • Mouse Overs
  • Mouse Outs
  • Specific Key Presses

If the element you are manipulating is the element that is listening, you can use the this keyword.

If the element you are manipulating is NOT the element that is listening, you can use the element's id.

What do we know about objects?

Objects have characteristics and things they can do. i.e, properties and methods.

We as devs can then hook in to these properties and methods to manipulate the DOM.
